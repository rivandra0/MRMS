# Material Request Management System  
**by Muhammad Rivandra**  

This application is used to handle material requests with two roles:  
- **Production_User**  
- **Warehouse_Admin**  

---

## Frontend
- Next.js  
- Bootstrap  
- DataTable  

---

## Backend
- Spring Boot 3  
  - JWT Authentication  
  - PostgreSQL  
  - Flyway (migration)  
  - MyBatis (mapper/ORM)  

---

**Detailed guides** are available in each folder:  
- `MRMS_FE` (Frontend)  
- `MRMS_API` (Backend)
